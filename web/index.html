How it works
Step 01

Sign in

Verify your phone number to get a pairing token and an API key.

Step 02

Install the iOS app

Paste your pairing token. PingClaw runs in the background and shares your location with your assistant when needed.

Step 03

Connect your agent

Add the configuration snippet to your agent. It can now answer questions about where you are.

In use

A quiet utility, not an app you stare at.

PingClaw runs in the background. When your AI assistant needs your location, it gets a single coordinate — accurate, current, scoped to you. No history, no map, no notifications.

Pairing token

Used by the PingClaw iOS app. Shown only at creation and rotation — the server stores only a hash. Rotating revokes the previous token.

Used by your AI agent. Shown only at creation and rotation — the server stores only a hash. Rotating breaks any agent using the previous key.
