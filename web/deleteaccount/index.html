Account deletion

Delete your account

You can permanently delete your PingClaw account and all associated data at any time. Deletion is immediate and irreversible.

What is deleted

  • Your account and user ID
  • All sign-in identities (Apple, Google)
  • All authentication tokens (pairing token, API key, web sessions)
  • Webhook configuration (URL and secret)
  • OpenClaw gateway configuration
  • Your cached location

After deletion, nothing remains on the server. Token hashes, identity links, and all configuration are permanently removed.

How to delete

From the app — open Settings, scroll to Account & data, tap "Delete all data", and confirm.

From the web — sign in to the dashboard, scroll to the "Delete account" section, and click "Delete account".

What happens after deletion

  • The app returns to the sign-in screen.
  • Any connected agents stop receiving location data immediately.
  • Your API key and pairing token stop working.
  • Webhook and OpenClaw deliveries stop.

There is no recovery. If you want to use PingClaw again, you will need to create a new account.
